leetcode
========
Solutions of some Leetcode problems

## How to run
For C++ solution:

```bash
g++ -std=c++11 ./src/example/example.cpp -o example.exe

clang-cl ./src/example/example.cpp -o example.exe
```

For C++ lint:
```bash
clang-tidy ./src/example/example.cpp -checks=*
```
For go solutin:
```bash
go run ./src/example/example.go
```
